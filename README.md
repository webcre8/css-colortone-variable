# css-colortone-variable
任意の色をもとにそのトーン違いの色を作るSass用の変数

var.scssには変数一覧が入っています。

サイトのメインカラー、サブカラー、アクセントカラーを表す変数に好みのカラーコードを入れるとそれぞれの色のトーン別っぽいの色を吐き出す変数を用意できます。

おまけでテキストカラー、ベースカラーも入れてあります。必要な色だけ使うと良いでしょう。

細かいことはブログ記事に書きます。
